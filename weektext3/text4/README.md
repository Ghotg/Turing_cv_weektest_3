# 题目4：Matplotlib 基础绘图

## 任务要求

请编写一个Python程序，使用 Matplotlib 完成以下绘图任务：

1. 创建 x 轴数据：从 0 到 10，包含 100 个点
2. 计算三组 y 轴数据：
   - y1 = x^2（二次函数）
   - y2 = x^3（三次函数）
   - y3 = 2^x（指数函数）
3. 在同一个图中绘制这三条曲线：
   - y1 使用红色实线，标签为 "y = x²"
   - y2 使用蓝色虚线，标签为 "y = x³"
   - y3 使用绿色点线，标签为 "y = 2^x"
4. 添加标题 "数学函数对比"
5. 添加 x 轴标签 "x"，y 轴标签 "y"
6. 添加图例和网格
7. 保存图像为 `functions_plot.png`
