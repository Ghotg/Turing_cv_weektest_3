# 题目3：Numpy 数组操作

## 任务要求

请编写一个Python程序，使用 Numpy 完成以下任务：

1. 创建一个 100x100 的 numpy 数组，数组元素为 0 到 255 之间的随机整数
2. 对数组进行归一化处理，使所有值的范围在 0 到 1 之间（使用最小-最大缩放方法）
3. 计算归一化后数组的均值和标准差，并打印输出
4. 将归一化后的数组保存到文件 `matrix.npy`

## 要求

- 使用 Numpy 库完成所有操作
- 正确实现最小-最大归一化
- 能够计算统计指标并保存数组

## 提示

- 使用 `np.random.randint()` 生成随机整数数组
- 最小-最大归一化公式：`(x - min) / (max - min)`
- 使用 `np.mean()` 和 `np.std()` 计算统计量
- 使用 `np.save()` 保存数组到 .npy 文件
