# 题目5：综合应用 - 图像边缘检测与分析

## 任务描述

本题是一个综合性题目，需要结合 OpenCV、Numpy 和 Matplotlib 完成图像边缘检测和数据分析任务。

## 任务要求

请编写一个Python程序，完成以下任务：

1. 读取一张图像文件 `image.jpg`
2. 将图像转换为灰度图
3. 使用高斯模糊对图像进行预处理（核大小 5x5）
4. 使用 Canny 边缘检测算法检测图像边缘，尝试两组不同的阈值：
   - 阈值组1：低阈值=50，高阈值=150
   - 阈值组2：低阈值=100，高阈值=200
5. 使用 Numpy 统计：
   - 计算每个边缘检测结果中边缘像素的数量
   - 计算边缘像素占图像总像素的百分比
   - 打印输出这些统计信息
6. 使用 Matplotlib 创建一个 2x2 的子图显示：
   - 左上：原始灰度图
   - 右上：高斯模糊后的图像
   - 左下：第一组阈值的边缘检测结果
   - 右下：第二组阈值的边缘检测结果
7. 保存显示结果为 `edge_detection_result.png`
8. 将第一组阈值的边缘检测结果保存为 `edges.jpg`

## 准备工作

请准备一张测试图片 `image.jpg`，放在题目文件夹中。建议使用包含明显边缘的图片（如建筑物、物体轮廓等）。

## 要求

- 综合运用 OpenCV、Numpy 和 Matplotlib
- 正确实现图像预处理和边缘检测
- 能够使用 Numpy 进行统计分析
- 代码要有清晰的注释

## 提示

### OpenCV 部分
- 使用 `cv2.cvtColor()` 转换为灰度图
- 使用 `cv2.GaussianBlur()` 进行高斯模糊
- 使用 `cv2.Canny()` 进行边缘检测

### Numpy 部分
- 使用 `np.sum()` 或 `np.count_nonzero()` 统计边缘像素
- 使用 `.shape` 获取图像尺寸计算总像素数
- 使用数学运算计算百分比

### Matplotlib 部分
- 使用 `plt.subplot()` 创建 2x2 子图
- 使用 `cmap='gray'` 显示灰度图像
- 添加适当的标题
