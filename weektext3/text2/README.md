# 题目2：OpenCV 颜色检测与轮廓识别

## 任务要求

请编写一个Python程序，完成以下图像处理任务：

1. 读取一张图像文件 `image.jpg`
2. 将图像转换为 HSV 颜色空间
3. 定义一个范围来检测红色区域
4. 创建一个二值掩码，用于标识红色像素，并保存为 `red_mask.jpg`
5. 找到红色区域的轮廓
6. 在原始图像上绘制这些轮廓的边界框
7. 将最终结果保存为 `result.jpg`

## 准备工作

请准备一张包含红色物体的测试图片，命名为 `image.jpg`，放在题目文件夹或 `images` 子文件夹中。

## 要求

- 使用 OpenCV (cv2) 库完成图像处理
- 正确使用颜色空间转换和颜色范围检测
- 能够找到轮廓并绘制边界框

## 提示

- 使用 `cv2.cvtColor()` 转换到 HSV 颜色空间
- 使用 `cv2.inRange()` 创建颜色掩码
- 使用 `cv2.findContours()` 查找轮廓
- 使用 `cv2.boundingRect()` 获取边界框坐标
- 使用 `cv2.rectangle()` 绘制矩形
