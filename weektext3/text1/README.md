# 题目1：OpenCV 图像基础处理

## 任务要求

请编写一个Python程序，完成以下图像处理任务：

1. 读取一张图像文件（假设文件名为 `image.jpg`）
2. 将图像转换为灰度图
3. 对灰度图应用高斯模糊（核大小为 5x5）
4. 将处理后的结果保存为 `processed.jpg`

## 准备工作

请在题目文件夹中创建一个 `images` 文件夹，并在其中放置一张测试图片 `image.jpg`。

## 要求

- 使用 OpenCV (cv2) 库完成图像处理
- 确保程序能够正确读取、处理和保存图像
- 代码要有适当的注释

## 提示

- 使用 `cv2.imread()` 读取图像
- 使用 `cv2.cvtColor()` 进行颜色空间转换
- 使用 `cv2.GaussianBlur()` 应用高斯模糊
- 使用 `cv2.imwrite()` 保存图像
